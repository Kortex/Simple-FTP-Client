Notes

  • Files will be signed with GPG key 0x5E442088ED632E87
  • These compilers should be relocatateable to any drirectory. However, I build them in /opt/cross
  • These compilers are only functional for kernel builds. They cannot be used to build userspace code (yet)
  • I currently haven't built toolchains for kernel architectures:
    1. c6x
    2. hexagon
    3. score
    4. unicore
    If you can point me at patches and/or documentation for building those architectures please do :)

Source

These compilers are built from GCC and Binutils sources from the FSF, for convenience that are mirrored here. For certain architectures patches may have been applied. Please check the patches directory. They are built using a modified version of the buildall scripts from Segher Boessenkool.
